# Quickstart: Targeted Assignments

## Testing Scenarios

### 1. Create Class-targeted Assignment
1. Log in as Teacher.
2. Go to Assignments.
3. Click "New Assignment".
4. Choose "Standard Class".
5. Select a class and fill details.
6. Verify students in that class see the assignment.

### 2. Create Batch-targeted Assignment
1. Log in as Teacher.
2. Go to Assignments.
3. Click "New Assignment".
4. Choose "Course Batch".
5. Select a Course and then a Batch.
6. Verify ONLY students in that specific batch receive a notification and see the assignment.

### 3. Display Validation
1. Verify the assignment card shows "Grade 10-A" for class-targeted ones.
2. Verify it shows "Course Name - Batch Name" for batch-targeted ones.

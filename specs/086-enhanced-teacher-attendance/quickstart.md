# Quickstart: Enhanced Teacher Attendance

## Development Setup
1. Ensure the teacher account you are using has at least one Course and one Class assigned.
2. In the Admin Panel, create a **Batch** for one of the courses and assign students to it.

## Testing Scenarios

### Scenario 1: Class-wise Attendance
1. Log in as Teacher.
2. Go to Attendance.
3. Select "Standard Class".
4. Select a class and verify students load.
5. Mark attendance and save.

### Scenario 2: Batch-wise Attendance
1. Log in as Teacher.
2. Go to Attendance.
3. Select "Course Batch".
4. Select a Course, then select a Batch.
5. Verify ONLY students in that batch load.
6. Mark attendance and save.
7. Reload and verify statuses are preserved.

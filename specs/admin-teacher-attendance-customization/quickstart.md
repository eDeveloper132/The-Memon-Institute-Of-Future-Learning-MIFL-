# Quickstart: Admin Teacher Attendance

## Verification Steps

### 1. Actual Time Viewing
1. Mark a teacher present via API or existing system.
2. Visit `teacher-attendance.html`.
3. Select current date.
4. Verify the log shows the precise check-in time.

### 2. Custom Time Entry
1. Click "Record Attendance".
2. Select a teacher from the list.
3. Choose a custom date (e.g., yesterday).
4. Set a custom time (e.g., 08:30 AM).
5. Save and verify the record appears in the history with the correct custom time.

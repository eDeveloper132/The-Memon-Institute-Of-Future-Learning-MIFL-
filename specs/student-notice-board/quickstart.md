# Quickstart: Student Notice Board

## Prerequisites
- Authenticated student account.
- Active notices created by admin with "students" audience or target class.

## Setup
1. Admin creates a notice via the Admin Dashboard.
2. Select "Students" as audience.
3. Choose a type (e.g., `exam`).
4. (Optional) Select a specific class.
5. (Optional) Pin the notice.

## Validation Scenarios

### Scenario 1: Dashboard Widget
1. Login as a student.
2. Observe the "Latest Notices" section.
3. Verify that pinned notices appear first.
4. Verify that the correct type icon is displayed.

### Scenario 2: Search & Filter
1. Navigate to the full Notice Board (`View All`).
2. Type a keyword in the search bar.
3. Verify the list updates instantly.
4. Select a category filter (e.g., "Holiday").
5. Verify only relevant notices are shown.

### Scenario 3: Attachments
1. Locate a notice with an attachment.
2. Click the download icon.
3. Verify the file opens/downloads correctly.
